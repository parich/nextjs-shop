#nvm (Node Version Manager)
- nvm ls-remote คำสั่งนี้เป็นการแสดงเวอร์ชัน node.js ออกมาทางหน้าจอ
- nvm install <version> คำสั่งนี้จะเป็นดาวน์โหลด node และ dependencies
- nvm use <node version> คำสั่งนี้เป็นการใช้ node version ใดใดก็ตามที่อยู่ในเครื่อง
